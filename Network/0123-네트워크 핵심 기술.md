# 네트워크 핵심 기술 (0123)

## 1. NAT & PAT

### 1) NAT (Network Address Translation)
*   **개요**: 사설 IP를 공인 IP로 변환하여 인터넷 통신을 가능하게 하거나 보안을 강화하는 기술.
*   **SNAT (Source NAT)**: 출발지 IP를 변경 (내부 -> 외부 접속 시).
*   **DNAT (Destination NAT)**: 목적지 IP를 변경 (외부 -> 내부 서버 접속 시, 로드밸런서 VIP -> Real IP).

### 2) PAT (Port Address Translation)
*   하나의 공인 IP를 여러 사설 IP가 공유하여 사용할 수 있도록 포트 번호를 매핑하는 기술 (NAPT).
*   가정용 공유기가 이 방식을 사용.

## 2. DNS (Domain Name System)

### 1) 구조
*   도메인 이름을 IP 주소로 변환하는 계층형 데이터베이스 시스템.
*   **Root(.)** -> **TLD(.com)** -> **Second Level(naver)** -> **Third Level(www)**.

### 2) 주요 레코드
*   **A**: 도메인 -> IPv4 주소.
*   **AAAA**: 도메인 -> IPv6 주소.
*   **CNAME**: 도메인 별칭 (Alias).
*   **MX**: 메일 서버 지정.
*   **NS**: 네임 서버 지정.
*   **PTR**: IP 주소 -> 도메인 (역방향).

### 3) GSLB (Global Server Load Balancing)
*   DNS 기반의 전역 로드밸런싱.
*   사용자의 위치나 서버의 상태(Health Check)를 고려하여 최적의 서버 IP를 응답.

## 3. DHCP (Dynamic Host Configuration Protocol)

### 1) 개요
*   IP 주소, 서브넷 마스크, 게이트웨이, DNS 정보를 클라이언트에게 자동으로 할당해주는 프로토콜.
*   **DORA 과정**: Discover -> Offer -> Request -> Acknowledge.
*   **포트**: Server UDP 67, Client UDP 68.
