# 네트워크 기초 이론 (0119)

## 1. 네트워크 프로토콜 및 모델

### 1) 프로토콜 (Protocol)
*   컴퓨터 간의 통신을 위한 규약 (약속).
*   **TCP/IP**: 현재 인터넷 표준 프로토콜 스택.
    *   **Application**: HTTP, FTP, SSH, DNS.
    *   **Transport**: TCP, UDP.
    *   **Internet**: IP, ICMP, ARP.
    *   **Network Access**: Ethernet.

### 2) OSI 7계층 (Reference Model)
1.  **Physical**: 전기 신호 전송 (Hub, Cable).
2.  **Data Link**: MAC 주소 기반 통신 (Switch).
3.  **Network**: 라우팅 및 논리적 주소 (IP) (Router).
4.  **Transport**: 신뢰성 있는 전송 및 포트 번호 (TCP/UDP) (Load Balancer).
5.  **Session**: 통신 세션 관리.
6.  **Presentation**: 데이터 표현 및 암호화/압축.
7.  **Application**: 사용자 인터페이스 및 응용 프로그램.

### 3) 캡슐화 (Encapsulation)
*   데이터 전송 시 상위 계층에서 하위 계층으로 내려가며 헤더를 붙이는 과정.
*   Data -> Segment (L4) -> Packet (L3) -> Frame (L2) -> Bit (L1).

## 2. 네트워크 장비 및 유형

### 1) 네트워크 유형
*   **LAN (Local Area Network)**: 근거리 통신망 (이더넷 기반).
*   **WAN (Wide Area Network)**: 원거리 통신망 (라우터 접속).
*   **VPN**: 공중망을 전용선처럼 사용하는 가상 사설망.

### 2) 주요 장비
*   **NIC (LAN카드)**: 물리적 연결 및 MAC 주소 보유.
*   **Hub (L1)**: 신호를 모든 포트로 단순 중계 (더미 허브).
*   **Switch (L2)**: MAC 주소를 학습하여 특정 포트로만 전송.
*   **Router (L3)**: 서로 다른 네트워크(LAN)를 연결 및 경로 설정.
*   **Load Balancer (L4/L7)**: 트래픽 부하 분산.

### 3) 통신 방식
*   **Unicast**: 1:1 통신 (가장 일반적).
*   **Broadcast**: 1:All 통신 (동일 네트워크 전체).
*   **Multicast**: 1:Group 통신 (특정 그룹).
*   **Anycast**: 1:Nearest 통신 (가장 가까운 대상).
