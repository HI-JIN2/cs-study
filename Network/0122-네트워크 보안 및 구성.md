# 네트워크 보안 및 구성 (0122)

## 1. 지능형 로드밸런싱 (ADC)

### 1) ADC (Application Delivery Controller)
*   **개요**: L7 로드밸런서의 진화형. 애플리케이션 프로토콜을 이해하고 최적화.
*   **기능**:
    *   **부하 분산**: L4/L7 로드밸런싱.
    *   **가속**: 캐싱, 압축, SSL Offloading.
    *   **보안**: WAF(Web Application Firewall) 기능 탑재 가능.

### 2) 시스템 확장 전략
*   **Scale Up (수직 확장)**: 개별 장비의 성능(CPU, RAM)을 높임. 비용 급증 및 한계 존재.
*   **Scale Out (수평 확장)**: 장비의 대수를 늘려 병렬 처리. 로드밸런서 필요. 클라우드 환경에 적합.

## 2. 네트워크 실습 환경 구성 (GNS3)

### 1) GNS3 개요
*   실제 Cisco IOS 이미지를 에뮬레이션하여 리얼한 네트워크 실습 환경 제공.
*   **Dynamips**: Cisco 라우터를 에뮬레이션하는 하이퍼바이저.

### 2) 라우터 및 스위치 기본 설정
*   **모드 전환**: User EXEC(`>`) -> Privileged EXEC(`enable`, `#`) -> Global Config(`conf t`, `(config)#`).
*   **인터페이스 설정**:
    ```text
    Router(config)# interface f0/0
    Router(config-if)# ip address 192.168.1.1 255.255.255.0
    Router(config-if)# no shutdown
    ```
*   **스태틱 라우팅**:
    ```text
    Router(config)# ip route [목적지 네트워크] [서브넷 마스크] [Next Hop IP]
    ```
*   **스위치 VLAN 설정**:
    ```text
    Switch(config)# vlan 10
    Switch(config-vlan)# name DATA
    Switch(config)# interface f0/1
    Switch(config-if)# switchport mode access
    Switch(config-if)# switchport access vlan 10
    ```
