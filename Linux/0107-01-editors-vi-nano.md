# Editors

## 1. 편집기

### 1) vi

- 특정 행으로 바로 이동

행번호G
:행번호
- 문서의 내용이 길 때 사용
- 코딩 시 많이 사용

- 내용 삭제

x, #x: 현재 위치의 글자를 삭제하고 #x는 글자 수를 지정하는 것이 가능
dw, #dw: 커서 위치의 단어를 삭제하고 #dw는 단어 수를 지정하는 것이 가능
dd #dd: 커서 위치의 행을 삭제하고 #dd는 행의 수를 지정하는 것이 가능
D: 커서 위치부터 행의 끝까지 삭제합니다.

- 명령 취소

u: 명령을 취소
U: 해당 행에서 한 모든 명령을 취소
:e!: 마지막으로 저장한 내용 이후의 것을 버리고 새로 작업합니다.

- **편집**:

yy, #yy: 커서가 위치한 행을 복사하는데 #yy는 행의 수를 지정
p: 커서가 위치한 행의 아래쪽에 붙여줍니다.
P: 행의 위에 붙여줍니다.
dd, #dd: 삭제하고 메모리에 저장

- **실습**:

- vim으로 새로운 파일인 exec.txt를 생성(vim exec.txt)

- 내용을 작성(i 나 a 나 o 키를 눌러서 삽입 모드로 진입해서 작성)
Good morning everyone.
Nice to meet you
I am linux beginner
Now introduce yourself

- 파일의 내용 저장(ESC를 누르고 저장만 할 때는 :w 저장하고 종료할 때는 :wq!)

- 파일을 열기(vim exec.txt)

- 커서를 3행으로 이동(명령모드에서 :3)

- 파일의 내용 중 beginner를 expert로 수정

- 2행의 meet 을 삭제

- 2행의 you 부터 끝까지 삭제

- 삭제한 내용 모두 복원

- 다시 저장

- 내용 검색

- 검색은 명령어 입력 줄인 마지막 행으로 이동해서 입력
- :가 아닌 / 나 ?를 눌러서 검색할 키워드를 입력

/문자열 - 문자열을 아래 방향으로 검색
?문자열 - 문자열을 위 방향으로 검색
n - 원래 찾던 방향으로 다음 문자열을 검색
N - 반대 방향으로 다음 문자열을 검색

- **치환**:

:s/문자열1/문자열2/ : 커서가 위치한 행에서 첫번째로 나오는 문자열1을 문자열2로 치환
:s/문자열1/문자열2/g : 파일 전체에서 나오는 문자열1을 문자열2로 치환
:<범위>s/문자열1/문자열2/ : 모든 각 행에서 첫번째로 나오는 문자열1을 문자열2로 치환
:<범위>s/문자열1/문자열2/g : 모든 각 행에서 나오는 문자열1을 문자열2로 치환
:<범위>s/문자열1/문자열2/gc : 모든 각 행에서 나오는 문자열1을 문자열2로 치환할 지 여부를 물음

:set ignorecase : 대소문자 구분하지 않고 변경
:%s/.*/\L&/g: 대문자를 소문자로 바꾸기
:%s/.*/\U&/g: 소문자를 대문자로 바꾸기

- 파일 읽어오기

:r 파일경로 : 지정한 파일을 현재 커서 위치에 삽입합니다.
:e 파일: 지정한 파일로 전환합니다.(현재 파일을 저장해야 합니다.)
:n: vi 시작 시 여러 파일을 지정한 경우 다음 파일로 작업을 이동합니다.

- shell 명령 수행

:!명령 : vi 작업을 중단하고 shell 명령을 수행하는데 vi로 돌아올려면 Enter
:sh : vi를 잠시나가서 shell 명령을 수행할 수 있는데 돌아올 때는 exit

- 기타 명령

CTRL + l: 현재 화면을 다시 출력
CTRL + g: 현재 커서 위치의 행 번호를 마지막 행에 출력
SHIFT + j: 현재 행과 아랫행을 연결해서 하나의 행으로 생성
.: 바로 직전의 명령 재수행

- vi 환경 설정

- 사용자 홈 디렉토리에 .exrc 파일에 설정
- 환경 변수 EXINIT에 지정
- vi의 마지막 행 모드에서 명령으로 실행

- 환경 설정 옵션

set nu        : 줄 번호 출력
set nonu        : 줄 번호 출력 안 함
set list        : 눈에 보이지 않는 문자 출력
set nolist        : 눈에 보이지 않는 문자 출력 하지 않음
set showmode    : 현재 모드 표시
set noshowmode    : 현재 모드 표시 하지 않음
set        : set으로 설정한 모든 vi 환경 설정 값을 출력
set all        : 모든 vi 환경 변수 와 현재 값을 출력
