# 서버 구축 및 프로그래밍 언어 (0115)

## 1. 프로그래밍 언어 환경 설정

### 1) C Programming
*   **컴파일러**: gcc (`sudo apt install gcc`).
*   **컴파일**: `gcc -o hello hello.c` -> `./hello`.
*   **Make**: 대규모 프로젝트 빌드 관리 도구 (`sudo apt install make`).
    *   `makefile`에 의존성 및 빌드 규칙 정의 후 `make` 명령 실행.

### 2) Java
*   **설치**: `sudo apt install openjdk-21-jdk`.
*   **확인**: `java -version`, `javac -version`.
*   **실행**: `javac Hello.java` -> `java Hello`.

### 3) Python
*   **설치**: 우분투에 기본 내장 (`python3 --version`).
*   **가상환경 (venv)**: 프로젝트별 패키지 격리.
    *   `python3 -m venv myenv`
    *   `source myenv/bin/activate`

### 4) Node.js
*   **설치**: nvm(Node Version Manager) 권장.
    *   `nvm install --lts`: 최신 LTS 버전 설치.
*   **프로젝트 초기화**: `npm init -y`.

### 5) Go
*   **설치**: 공식 사이트에서 타르볼 다운로드 후 `/usr/local`에 압축 해제 및 PATH 설정.
*   **실행**: `go run main.go`.

## 2. 서버 구축

### 1) 원격 접속 (Remote Access)
*   **Telnet**: 평문 전송으로 보안 취약. 테스트용으로만 제한적 사용.
*   **SSH (Secure Shell)**: 암호화된 원격 접속 프로토콜 (포트 22).
    *   **설치**: `sudo apt install openssh-server`.
    *   **서비스 제어**: `sudo systemctl start ssh`.
    *   **키 기반 인증**:
        1.  클라이언트에서 키 생성: `ssh-keygen -t rsa`.
        2.  공개키 전송: `ssh-copy-id user@server_ip`.
    *   **SCP**: 파일 전송. (`scp file user@host:/path`).

### 2) 데이터베이스 서버
*   **MariaDB**: MySQL과 호환되는 오픈소스 RDBMS.
    *   `sudo apt install mariadb-server`.
    *   `sudo mysql_secure_installation`: 보안 설정.
    *   외부 접속 허용 시 `bind-address = 0.0.0.0` 설정 필요.
*   **MongoDB**: NoSQL 문서 지향 데이터베이스.
    *   `sudo apt install mongodb-org`.
    *   `mongosh`: 셸 접속.
*   **Redis**: In-Memory Key-Value 저장소 (캐시, 세션 관리 등).
    *   `sudo apt install redis-server`.
    *   `redis-cli`: 클라이언트 접속.
