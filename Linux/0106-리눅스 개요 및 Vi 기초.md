# 리눅스 개요 및 Vi 기초 (0106)

## 1. 리눅스(Linux) 개요

### 1) Linux란?
*   리누스 토발즈(Linus Torvalds)가 MINIX 교육용 OS를 참조하여 개발.
*   **유닉스(UNIX) 계열**의 운영체제: Linux = Linus + UNIX.
*   **Kernel(커널)**: 운영체제의 핵심. 프로세스, 메모리, 파일 시스템, 장치 관리 등 수행. (대부분 C언어로 작성)
*   **Shell(쉘)**: 명령어 해석기. 유저의 명령을 커널에 전달.
*   **배포판 계열**:
    *   **데비안(Debian)**: 우분투(Ubuntu), 리눅스 민트 등.
    *   **레드햇(RedHat)**: 페도라(Fedora), CentOS(Rocky), RHEL.
    *   **슬랙웨어(Slackware)**: SuSE 등.

### 2) 가상화(Virtualization)
*   하나의 물리적 컴퓨터 자원(CPU, Memory 등)을 논리적으로 분할하여 여러 운영체제를 실행하는 기술.
*   **가상 머신 (VM)**: 하이퍼바이저 위에서 Guest OS를 실행. (VirtualBox, VMware, KVM 등)
*   **컨테이너 (Container)**: OS 수준의 가상화. (Docker 등)

### 3) 우분투(Ubuntu) 설치 및 접속
*   **설치 방법**: VirtualBox 등의 가상 머신에 ISO 이미지(Server/Desktop)를 이용하여 설치.
*   **SSH 접속**:
    *   `openssh-server` 설치: `sudo apt install openssh-server`
    *   서비스 시작: `sudo systemctl start ssh`
    *   접속: `ssh 계정@호스트IP -p 포트`

## 2. 기본 명령어와 시스템 종료

### 1) 종료 및 재부팅
*   **종류**: `poweroff`, `halt -p`, `init 0`
*   **shutdown**:
    *   `shutdown -P now`: 즉시 종료.
    *   `shutdown -r now`: 즉시 재부팅.
    *   `shutdown -k +10`: 10분 뒤 종료 메시지만 전송(실제 종료 X).
*   **재부팅**: `reboot`, `init 6`.

### 2) Run Level
| Level | Mode | 설명 |
| :--- | :--- | :--- |
| 0 | Power Off | 종료 |
| 1 | Rescue | 복구 모드 (단일 사용자) |
| 3 | Multi-User | 텍스트 모드 (CLI) |
| 5 | Graphical | 그래픽 모드 (GUI) |
| 6 | Reboot | 재부팅 |

### 3) Shell 기본 기능
*   **자동 완성**: `Tab` 키 (1번: 자동 완성, 2번: 목록 표시).
*   **History**: `history` 명령, `!!`(직전 명령), `!번호`(해당 번호 명령).
*   **단축키**:
    *   `CTRL + c`: 강제 종료.
    *   `CTRL + l`: 화면 지우기 (`clear`).
    *   `CTRL + a`: 라인 맨 앞으로 이동.
    *   `CTRL + e`: 라인 맨 뒤로 이동.

## 3. Vi / Vim 편집기

### 1) Vi 편집기의 모드
Vi는 **모드형 편집기**로, 3가지 주요 모드가 있습니다.
1.  **명령 모드 (Command Mode)**: 처음 실행 시 상태. 커서 이동, 복사, 붙여넣기, 삭제 등 수행.
2.  **입력 모드 (Insert Mode)**: 텍스트 입력 상태.
3.  **마지막 행 모드 (Last Line Mode)**: 저장, 종료, 검색, 치환 등 수행. (`:`으로 진입)

### 2) 모드 전환
*   **명령 -> 입력**:
    *   `i`: 커서 앞(현재 위치) 입력.
    *   `a`: 커서 뒤 입력.
    *   `o`: 현재 줄의 다음 줄(아래)에 입력.
*   **입력 -> 명령**: `ESC` 키.
*   **명령 -> 마지막 행**: `:`, `/`, `?`.

### 3) 주요 명령어 (명령 모드)
| 키 | 설명 |
| :--- | :--- |
| `dd` | 한 줄 삭제 (잘라내기) |
| `yy` | 한 줄 복사 |
| `p` | 붙여넣기 |
| `u` | 실행 취소 (Undo) |
| `CTRL + r` | 재실행 (Redo) |
| `gg` | 문서 맨 처음으로 이동 |
| `G` | 문서 맨 끝으로 이동 |
| `^` (Home) | 행의 처음으로 이동 |
| `$` (End) | 행의 끝으로 이동 |

### 4) 저장 및 종료 (마지막 행 모드)
*   `:w`: 저장.
*   `:q`: 종료.
*   `:wq`: 저장 후 종료.
*   `:q!`: 저장하지 않고 강제 종료.
*   `:set nu`: 라인 번호 표시.
