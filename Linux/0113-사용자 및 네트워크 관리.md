# 사용자 및 네트워크 관리 (0113)

## 1. 사용자 관리

### 1) 사용자 및 그룹 관리 파일
*   `/etc/passwd`: 사용자 계정 정보 (아이디, UID, GID, 홈 디렉토리, 쉘).
*   `/etc/shadow`: 암호화된 비밀번호 및 패스워드 에이징 정보 (Root만 접근 가능).
*   `/etc/group`: 그룹 정보.
*   `/etc/skel`: 계정 생성 시 홈 디렉토리에 복사될 기본 파일들이 있는 디렉토리.

### 2) 사용자 관리 명령어
*   **useradd**: 사용자 생성.
    *   `useradd -m -s /bin/bash user1`: 홈 디렉토리 생성 및 쉘 지정.
*   **passwd**: 비밀번호 설정.
    *   `passwd user1`: user1의 비밀번호 변경.
    *   `passwd -l user1`: 계정 잠금.
*   **usermod**: 사용자 정보 수정.
    *   `usermod -aG sudo user1`: user1을 sudo 그룹에 추가.
*   **userdel**: 사용자 삭제.
    *   `userdel -r user1`: 홈 디렉토리까지 함께 삭제.

### 3) 그룹 관리 명령어
*   **groupadd**: 그룹 생성.
*   **groupmod**: 그룹 수정.
*   **groupdel**: 그룹 삭제.
*   **gpasswd**: 그룹 관리 (사용자 추가/제거).
    *   `gpasswd -a user1 group1`: user1을 group1에 추가.

### 4) 권한 상승 (sudo)
*   일반 사용자가 root 권한을 빌려 명령을 실행.
*   `/etc/sudoers` 파일에 설정 (반드시 `visudo` 명령어로 편집).

## 2. 네트워크 관리 기초

### 1) 네트워크 기본 개념
*   **IP Address**: 컴퓨터 고유의 식별 주소 (IPv4: 32bit).
*   **MAC Address**: NIC(랜카드)의 고유 하드웨어 주소 (48bit).
*   **Port**: 서비스(프로세스)를 구분하는 번호 (HTTP: 80, SSH: 22).

### 2) 네트워크 관리 도구
*   **NetworkManager**: 우분투의 기본 네트워크 관리 데몬. (`nmcli` 명령 사용)
*   **nmcli**:
    *   `nmcli general status`: 네트워크 상태 확인.
    *   `nmcli dev status`: 네트워크 장치 상태 확인.
    *   `nmcli con show`: 연결 정보 확인.
*   **ip**:
    *   `ip addr`: IP 주소 확인 (`ifconfig` 대체).
    *   `ip route`: 라우팅 테이블 확인.

### 3) 쿼터 (Quota)
*   사용자의 디스크 사용량 및 파일 개수를 제한하는 기능.
*   `/etc/fstab`에 `usrquota`, `grpquota` 옵션 추가 후 마운트.
*   `edquota -u user1`: 사용자 쿼터 설정 편집.
