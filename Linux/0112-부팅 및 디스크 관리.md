# 부팅 및 디스크 관리 (0112)

## 1. 부팅 및 서비스 관리

### 1) systemd
*   **개요**: 기존 init 프로세스를 대체하는 최신 리눅스 서비스 관리자. 병렬 실행을 지원하여 부팅 속도가 빠름.
*   **Unit**: systemd가 관리하는 리소스 단위 (Service, Target, Mount, Socket 등).
*   **주요 명령어 (systemctl)**:
    *   `systemctl start [서비스명]`: 서비스 시작.
    *   `systemctl stop [서비스명]`: 서비스 중지.
    *   `systemctl restart [서비스명]`: 서비스 재시작.
    *   `systemctl enable [서비스명]`: 부팅 시 자동 시작 설정.
    *   `systemctl status [서비스명]`: 서비스 상태 확인.
    *   `systemctl list-units --type=service`: 실행 중인 서비스 목록.

### 2) 시스템 종료 및 재부팅
*   **종료**:
    *   `shutdown -h now`
    *   `poweroff`
    *   `init 0`
*   **재부팅**:
    *   `reboot`
    *   `init 6`

### 3) GRUB (Boot Loader)
*   리눅스 커널을 메모리에 로딩하는 역할.
*   설정 파일: `/boot/grub/grub.cfg` (직접 수정 권장 안 함).
*   설정 변경: `/etc/default/grub` 파일 수정 후 `sudo update-grub` 실행.

### 4) Namespace & Cgroup (컨테이너 기술 기반)
*   **Namespace**: 프로세스를 격리 (PID, Network, Mount 등 독자적인 공간 제공).
*   **Cgroup (Control Group)**: 프로세스의 자원(CPU, Memory) 사용량을 제한 및 모니터링.

## 2. 파일 시스템 및 디스크 관리

### 1) 주요 파일 시스템
*   **ext4**: 리눅스 표준 파일 시스템. 저널링 지원.
*   **XFS**: 대용량 파일 처리에 최적화된 고성능 파일 시스템 (CentOS/RHEL 기본).
*   **가상 파일 시스템**:
    *   `/proc`: 커널 및 프로세스 정보 (메모리 상에 존재).
    *   `/sys`: 하드웨어 장치 정보.
    *   `/dev`: 장치 파일.

### 2) 디스크 관리 및 파티셔닝
*   **fdisk**: 파티션 생성 및 관리 도구 (MBR 방식).
    *   `fdisk -l`: 디스크 목록 확인.
    *   `n`: 새 파티션 생성.
    *   `d`: 파티션 삭제.
    *   `w`: 저장.
*   **mkfs**: 파일 시스템 생성 (포맷).
    *   `mkfs.ext4 /dev/sdb1`: ext4로 포맷.

### 3) 마운트 (Mount)
*   물리적인 장치(디스크 파티션, USB 등)를 디렉토리에 연결하는 작업.
*   **명령어**:
    *   `mount [장치] [마운트포인트]`: 마운트.
    *   `umount [마운트포인트]`: 해제.
*   **자동 마운트 (/etc/fstab)**: 부팅 시 자동으로 마운트되도록 설정하는 파일.

### 4) LVM (Logical Volume Manager)
*   여러 물리 디스크를 하나의 논리적 볼륨으로 묶어 유연하게 관리.
*   **구성 요소**:
    *   **PV (Physical Volume)**: 물리적 디스크.
    *   **VG (Volume Group)**: PV들의 집합.
    *   **LV (Logical Volume)**: 사용자가 실제로 사용하는 논리적 파티션.

### 5) 디스크 사용량 확인
*   **df (Disk Free)**: 파일 시스템 별 디스크 사용량 확인. (`df -h`)
*   **du (Disk Usage)**: 디렉토리 별 용량 확인. (`du -sh [디렉토리명]`)
*   **fsck**: 파일 시스템 무결성 점검 및 복구.
